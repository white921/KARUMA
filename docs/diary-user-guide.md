# 📖 日記機能の使い方

日記フォーラムに、自分専用の通常日記またはVIP日記を作成できます。作成後は、作られたスレッドにそのまま投稿して日記を続けてください。

## 使い始める前に

- 日記を作成するには、教団銀行の口座が必要です。
- サブアカウントは、日記の作成・VIP化・再開を行えません。
- 日記は1人につき1つです。すでに利用中の日記がある場合、新しい日記を追加では作成できません。

## 作り方

1. 日記パネルで、作りたい種類のボタンを押します。
2. タイトルと最初の本文を入力します。本文は空欄でも作成できます。
3. 内容を確認して、表示される確認ボタンを押します。
4. 作成された日記スレッドを開き、以後はそのスレッドへ投稿します。

> 本文を空欄にした場合は「日記を作成しました。」と投稿されます。

## 日記の種類と料金

### 🔒 通常日記

- 料金：5,000krm
- スレッド名には🔒が付きます。
- 作成者本人と、紐づけ済みのサブアカウントの投稿だけを残す日記です。
- ほかの人が投稿した場合、Botが投稿を削除します。

> Discordの仕様上、投稿自体を完全に事前ブロックするのではなく、投稿後にBotが削除する方式です。

### VIP日記

- 料金：50,000krm
- 通常日記と異なり、Botによる他ユーザー投稿の削除は行われません。
- 閲覧・投稿できる範囲は日記フォーラムのDiscord権限に従います。

### 無料対象

本メン・鯖主・管理者の各ロールを持つ人は、日記の作成・VIP化を無料で利用できます。

## 通常日記をVIP日記へ変更する方法

通常日記を作成済みで、日記が開いている場合は、日記パネルの**「アップグレード」**を押します。

- 料金：45,000krm（通常日記との差額）
- 新しいタイトルを入力できます。
- 本文を入力した場合は、日記スレッドへ新しい投稿として追加されます。
- VIP日記を通常日記へ戻す操作は、日記パネルにはありません。

## 3日間投稿がない場合

作成者本人または紐づけ済みのサブアカウントから**3日間投稿がない**日記は、毎日0:10ごろ（日本時間）に自動でクローズされます。

クローズ後に再開したい場合は、日記パネルから通常日記またはVIP日記の作成ボタンを押してください。タイトルと本文を入力して確定すると、同じ日記スレッドを再開できます。

> ほかの人の投稿は、通常日記の継続判定には含まれません。

## 注意事項

- 作成・再開・VIP化の前に、必要なkrm残高があるか確認してください。
- 一度VIP日記にすると、アップグレードボタンでは更新できません。
- 通常日記を利用中にVIPへ変更したいときは、通常日記の作成ボタンではなく**「アップグレード」**を使ってください。
